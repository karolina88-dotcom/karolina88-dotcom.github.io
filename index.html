HTML (HyperText Markup Language)

HTML je značkovací jazyk, který tvoří základ každé webové stránky. Poskytuje strukturu a obsah stránky pomocí tagů a prvků.

Základní struktura HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Název stránky</title>
  </head>
  <body>
    <h1>Nadpis</h1>
    <p>Odstavec textu</p>
  </body>
</html>

Běžné HTML tagy:

  • <h1> až <h6> - Nadpisy
  • <p> - Odstavec
  • <a> - Odkaz
  • <img> - Obrázek
  • <div> - Kontejner
  • <button> - Tlačítko
  • <form> - Formulář
  • <ul>, <ol>, <li> - Seznamy
🎨

CSS (Cascading Style Sheets)

CSS je jazyk pro popis stylu a vzhledu webových stránek. Umožňuje vám barvit, měnit velikost a zarovnávat HTML prvky.

Základní syntaxe CSS:

selektor {
  vlastnost: hodnota;
  vlastnost: hodnota;
}

Příklad:

h1 {
  color: #333;
  font-size: 32px;
  text-align: center;
  margin: 20px 0;
}

Typy selektorů:

  • Prvek - p, div, h1
  • Třída - .moje-trida
  • ID - #moje-id
  • Atribut - [type="text"]
  • Pseudo-třídy - :hover, :active, :focus

Vlastnosti CSS:

  • color - Barva textu
  • background-color - Barva pozadí
  • font-size - Velikost písma
  • margin - Vnější odsazení
  • padding - Vnitřní odsazení
  • width, height - Šířka a výška
  • display - Typ zobrazení (block, inline, flex, grid)
⚙️

JavaScript

JavaScript je programovací jazyk, který přináší interaktivitu na webové stránky. Běží v prohlížeči a umožňuje reagovat na akce uživatele.

Základní syntaxe:

let jmeno = "Petr";
function pozdrav() {
  console.log("Ahoj " + jmeno);
}
pozdrav();

Co umožňuje JavaScript:

  • Ověřování formulářů
  • Animace a efekty
  • Práce s událostmi (kliknutí, najetí myší)
  • Dynamická změna obsahu stránky
  • Komunikace se serverem (AJAX)
  • Správa dat v prohlížeči

Příklad interaktivity:

document.getElementById("tlacitko").addEventListener("click", function() {
  alert("Klikli jste na tlačítko!");
  document.body.style.backgroundColor = "lightblue";
});
